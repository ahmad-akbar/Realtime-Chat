# Realtime-Chat
Adalah sebuah web chat-application sederhana dan realtime. Disamping itu realtime-chat ini sudah menggunakan database untuk penyimpanan message, room chat dan name user.

# Local Deploy

- Masuk ke folder Server > cd server
